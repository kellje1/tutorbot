# Project 2 - Web App

## Project Overview
TutorBot is an online platform that helps users get tutoring services online. As most tutoring centers are not always avaiable 24/7, TutorBot provides a valuable resource for students seeking academic support. 

## Features
Some of the features I've included are the home page, which talks about the webpage and includes the Gemini AI, the search page, where you could check out the subjects that you want to most help in, the signup page, where the user becomes a member of the website and puts their name, username, passwod, and the login page, where the user logs in only if they're signed up. 

## How to Run the Server
To run the server, for the JavaScript file, you would type in node tutor.js and it will tell you that the server is running on http//localhost:4000. 


## The name of the API
The API I've used is Google Gemini API. As you can see, it's been having some errors, but if I were to continue it, I would try to fix the issue with the API key and its connection. 

## Database Schema
For the database schema, I've used MongoDB. It will basically store user's information since they're signed into the website. It includes their first and last name, email, username and password.
